Tipos de juicio

Para poder obtener los datos correctos del problema y la ayuda que desea adquirir es conveniente que antes de seleccionar el tipo de caso al cual pertenece su problema lea la siguiente informacion:


Tipos de juicio: Penal En este caso, el juicio penal es el que se produce cuando se ha cometido algún delito o falta como por ejemplo robos, asesinatos, daños importantes, falsificaciones, tráfico de drogas, etc.

Tipos de juicio: Contencioso-administrativo Este tipo de juicio se refieren a reclamaciones entre particulares y la Administración (central, autonómica o local). Para poder optar a un juicio contecioso-administrativo se deben agotar primero todos los trámites administrativos previos. Su desarrollo es similar al del juicio civil.

Tipos de juicio: laboral Se producen cuando se reclaman asuntos relacionados con los derechos derivados del trabajo y la Seguridad Social como por ejemplo despido, sanciones laborales, jubilaciones, orfandad…
